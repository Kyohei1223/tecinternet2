# Quiz Frontend - Exame Preparation

Este projeto contém um Quiz interativo para preparação de exames de Frontend e Fullstack.

## Como usar
Basta abrir o ficheiro `index.html` num navegador.

## Deploy
Este projeto está pronto para ser alojado na Vercel.
1. Instale a Vercel CLI: `npm i -g vercel`
2. Corra `vercel` na pasta deste projeto.
OU
1. Faça push para o GitHub.
2. Importe o projeto no dashboard da Vercel.
